Решил создать пет проект - простенького ai-агента с функциями семейного доктора. Такое желание, конкретно, что он должен делать пока без понятия.

Начну с построения архитектуры:
1. Интерфейс: реализую через браузер. Чат с моделью типа облачного deepseek
2. LLM-Агент: модель. Получает запрос, решает, какие инструменты (функции) выполнить и формулирует ответ. Конкретно какая модель будет, решу по ходу дела. Желательно конечно заточенную на медицину и понимающую русский язык.
3. Инструменты (Tools): Это функции, которые агент должен выполнять. Какие понадобятся функции, также пока не знаю.
4. Векторная база данных (Qdrant): хранилище цифровых данных, по ним агент будет проводить семантический поиск. Учитывая что документов у меня будет 2 excel файла: один с почти ежедневными измеряемыми и вручную заносимые в этот файл показателями типа вес, давление  и другое. Второй файл содержит результаты анализов типа ОАК, биохимического и тому подобное.

Первую функцию наверное реализую: def make_plot(), чтобы когда робот Вася (агент) получит приказ нарисовать график моего веса, то вызывал эту функцию.
